# Time Suite Pro

Time Suite Pro es una aplicación web que te ayuda a gestionar el tiempo de manera eficiente. Incluye tres herramientas principales:

- **Relojes Mundiales**: Consulta la hora actual en varias ciudades del mundo.
- **Cronómetro**: Mide el tiempo en intervalos precisos con la opción de iniciar, pausar y reiniciar.
- **Temporizador**: Configura un temporizador con horas, minutos y segundos, y recibe una alarma cuando llegue a cero.

### Características:

- **Multilenguaje**: Disponible en inglés, español y alemán.
- **Diseño moderno**: Interfaz limpia y oscura, ideal para ambientes con poca luz.
- **Fácil de usar**: Funciones sencillas para gestionar el tiempo de manera eficiente.

## Uso

1. **Relojes Mundiales**: Consulta la hora en ciudades como Berlín, Madrid, Casablanca y Moscú.
2. **Cronómetro**: Haz clic en "Iniciar" para empezar a medir el tiempo y en "Detener" para pausarlo.
3. **Temporizador**: Configura el tiempo deseado y haz clic en "Iniciar". La alarma sonará cuando el temporizador termine.

¡Mantén el control de tu tiempo con **Time Suite Pro**!
